CSS Shaders

In a nutshell:
Allows you to manipulate pixels for a more "cinematic effect" using different types of CSS Shader types. A few examples fpimd were: book.vs, distort.vs, wobble.vs, etc.

** Webkit is still in development.

For more info:
http://dvcs.w3.org/hg/FXTF/raw-file/tip/custom/index.html
  • Using different types of "shaders" you can manipulate pixels/images.
  • Images are manipulated by using the "vertex mesh"
  • Vertex mesh defines the vertices/corners of the images.
  • The shaders are then applied to the image and manipulate the image by referring to the vertex mesh."

** See source of this page for a brief example of how CSS Shaders are implemented.
